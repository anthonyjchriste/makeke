    Welcome to Makeke!

    What is Makeke?

    Makeke is the Hawaiian word for market. This web application hopes to create a market that allows members of within small communities to exchange books.

    Makeke was written using the Play Framework and Twitter Bootstrap.

    What features does Makeke support?

    For a quick (~8 min) overview of the current features supported in Makeke, please watch my webcast about Makeke.

    Separate Accounts

    Makeke allows users to create their own separate account to store offers and requests.

    Update Account

    Users can update their account information (change e-mails, passwords, etc).

    Book Explorer

    Users can view offers and requests from all other users in the system using the Book Explorer. The book explorer also allows partial filtering of results. Currently, only filtering of book titles is enabled, but I would like to get the rest of the fields integrated as well in the future.

    Active Requests / Offers

    Management of a user's offers and requests. Users can view their current active requests and offers, add new requests and offers, and manage existing requests and offers from this screen.

    New Requests / Offers

    Users can create new requests and offers. If a book is already in the system, a user can select that book and the title, edition, and isbn will be copied to the correct fields.

    Editing Requests / Offers

    Existing requests and offers can be edited or deleted.

    Please Note

    You will need to create an account on Makeke before interacting with the system.
